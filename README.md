# SmartSchoolHub

SmartSchoolHub is a comprehensive web-based school portal and computer-based testing (CBT) platform developed by **HillCity Solutions**. It is designed to streamline administrative tasks and improve assessment processes for secondary schools.

## Features
- **Student Management:** Secure portal for student data.
- **CBT Platform:** Efficient, digital examination system for students.
- **Administrative Tools:** Built-in features to manage school operations effectively.

## Tech Stack
- **Framework:** Next.js (App Router)
- **Language:** TypeScript
- **Styling:** Tailwind CSS
- **Backend/Database:** Supabase
